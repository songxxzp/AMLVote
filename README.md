# 学术作品投票平台 - 使用说明

## 🎯 系统功能

### 1. 作品提交
- **必填信息**：作品标题、作者姓名、学号、邮箱地址
- **可选信息**：合作作者及其学号、作品描述、摘要、关键词
- **文件上传**：支持 PDF、Word、PPT 等格式
- **作品类型**：学术论文、学术海报、演示项目

### 2. 投票系统
- **身份验证**：使用学号和姓名进行身份验证
- **投票限制**：每位同学最多可以投 5 票
- **防重复投票**：同一作品只能投一次票
- **实时显示**：显示剩余票数

### 3. 作品展示
- **作品列表**：展示所有提交的作品
- **搜索过滤**：按类型、关键词、作者搜索
- **排序功能**：按投票数或时间排序
- **排行榜**：显示 TOP 10 最受欢迎的作品

## 📋 使用流程

### 提交作品
1. 点击"上传作品"按钮
2. 填写作品信息（*为必填项）
3. 上传相关文件（可选）
4. 点击"提交作品"

### 参与投票
1. 在页面顶部输入您的学号和姓名
2. 系统会自动显示您的剩余票数
3. 浏览作品列表，选择喜欢的作品
4. 点击"投票"按钮进行投票
5. 每人最多5票，投完即止

### 查看结果
1. 切换到"排行榜"标签
2. 查看当前最受欢迎的作品
3. 获票最多的作品将有机会上台展示

## 🔧 技术特性

- **学号认证**：基于学号的唯一身份识别
- **投票限制**：每人5票的公平投票机制
- **实时更新**：投票结果实时显示
- **文件管理**：安全的文件上传和存储
- **响应式设计**：支持手机、平板、电脑

## 📝 注意事项

1. **学号唯一性**：每个学号只能对应一个用户
2. **投票公平性**：禁止刷票，每人限5票
3. **作品原创性**：请确保提交的作品为原创
4. **信息准确性**：请填写真实的学号和联系信息

## 🎁 评选规则

- 投票结束后，获票数最多的前X名作品将获得上台展示机会
- 展示时间：最后一周
- 具体展示顺序将根据投票结果确定
